# Practical Scope Guardrails

## Current Active Scope

ProjectRAG's current active scope is limited to the local-first GraphRAG MVP foundation:

- FastAPI
- LangGraph
- PostgreSQL + pgvector
- GraphDB
- Ollama
- Document ingestion
- Vector retrieval
- Graph retrieval
- Hybrid answer generation
- Validation
- Tool governance in read-only mode

## Delayed Scope

The following capabilities are explicitly delayed and are not part of the current active MVP scope:

- autonomous execution
- Terraform apply
- Kubernetes deployment
- AGII
- recursive evolution
- artificial CEO/CTO/COO production logic
- cloud resource modification

## Stability Rule

Any future feature must pass MVP stability first.

This means future work should not bypass the core requirements for local reliability, tests, validation, RBAC/governance, auditability, and safe read-only defaults.
